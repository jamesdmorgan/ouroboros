Ouroboros
=========

Ouroboros is a simple wrapper for the eventstore management APIs, designed for managing permissions on streams. It is *not* a fully-featured eventstore client.

Features
--------

 - Create and manage users, groups, and streams
 - Post events to Eventstore from files, command line args, or STDIN
 - Handy-dandy command-line client
 - Ansible support incoming


Installation
-------------

Install Ouroboros by running

    pip install es-ouroboros
